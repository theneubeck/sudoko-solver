# sudoko-solver
A naive/brute force sudoku solver

## Usage

Use the empty_game-template to fill in your game, save it in for example `game.txt` then run

```bash
> cat game.txt | python solve.py

|8|9|3|4|2|7|6|1|5|
|2|4|6|1|8|5|7|9|8|
|5|7|1|7|6|3|2|4|8|
|3|5|2|6|5|4|8|7|1|
|1|5|9|7|3|8|4|6|2|
|4|8|7|2|1|6|5|3|3|
|6|1|5|3|4|2|1|8|7|
|6|3|8|5|4|6|1|2|7|
|9|2|4|8|7|1|3|5|6|
```
